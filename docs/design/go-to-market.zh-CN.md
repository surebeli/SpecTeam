# PhoenixTeam 推广方案 (Go-to-Market)

## 商业模式

| 层级 | 内容 | 价格 |
|------|------|------|
| Freemium | 开源核心（MCP Server + 基本同步脚本） | 免费 ≤5 成员自托管 |
| Team Pro | Dashboard、高级仲裁、审计 | $15/人/月 |
| Enterprise | 自定义（私有化 + Yunxin 深度集成 + 支持） | 年付 + 专业服务 |

## 推广阶段

### Awareness（0-3 月）

- 开源 MCP 核心到 GitHub
- Zhihu/CSDN/Dev.to 系列文章《多 AI 工具大战的终结》
- Demo 视频（5 个 Agent 30 分钟收敛）

### Acquisition（3-6 月）

- Beta 邀请制（Claude/Cursor 重度用户）
- 网易云信生态推广（客户清单、联合 Webinar）
- X / 小红书 / 微信视频号案例

### Conversion & Retention

- 落地 Page 免费试用
- 社区（Discord/微信群）
- 合作伙伴（Cursor 生态、IDE 厂商、NetEase 销售团队）
